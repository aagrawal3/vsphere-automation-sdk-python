Enum - HostSgxInfoSgxStates(vim.host.SgxInfo.SgxStates)

Since
vSphere API 7.0


Enum Description

Host SGX states.

Enum Constants

NameDescription
disabledBIOS SGX is disabled in the BIOS.
disabledCFW101 SGX is disabled because CPU erratum CFW101 is present.
disabledCPUMismatch SGX is disabled due to a mismatch in the SGX capabilities exposed by different CPUs.
disabledMaxEPCRegs SGX is disabled because the host exceeds the maximum supported number of EPC regions.
disabledNoFLC SGX is disabled because the CPU does not support FLC.
disabledNUMAUnsup SGX is disabled because the host uses NUMA, which is not supported with SGX.
enabled SGX is enabled.
notPresent SGX is not present in the CPU.
