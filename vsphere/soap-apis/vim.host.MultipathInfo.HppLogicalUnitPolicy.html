Data Object Description

The HostMultipathInfoHppLogicalUnitPolicy data object describes a multipathing policy for a HPP claimed logical unit and its configuration.

Properties

Name Type Description
bytes*xsd:long

Byte count on the paths will be used as criteria to switch path for the device. Allowed values 1 to (100*1024*1024) Default Value 10*1024*1024
iops*xsd:long

IOPS count on the paths will be used as criteria to switch path for the device. Allowed values 1 to 10000 Default Value 1000
latencyEvalTime*xsd:long

This value can control at what interval (in ms) the latency of paths should be evaluated. Allowed values 10000 to (300 * 1000) in ms Default Value 30 * 1000
path*xsd:string

The preferred path for the given device. If no prefered path is specified by the user, algorithem at ESX side will choose the random possible path.
samplingIosPerPath*xsd:long

This value will control how many sample IOs should be issued on each path to calculate latency of the path. Allowed values 16 to 160 Default Value 16
