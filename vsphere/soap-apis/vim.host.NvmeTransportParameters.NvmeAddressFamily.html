Enum - HostNvmeTransportParametersNvmeAddressFamily(vim.host.NvmeTransportParameters.NvmeAddressFamily)

Since
vSphere API 7.0


Enum Description

This enum specifies the supported address families for NVME over Fabrics.

For details, see:

  • "NVM Express over Fabrics 1.0", Section 5.3, Figure 34, "Discovery Log Page Entry"
