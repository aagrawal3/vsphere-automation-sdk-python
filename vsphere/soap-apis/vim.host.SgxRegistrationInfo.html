Data Object Description

Data object describing SGX host registration information.

Properties

Name Type Description
biosError*xsd:int

BIOS error related to SGX host registration. Set only if SGX registration status is incomplete.
lastRegisteredTime*xsd:dateTime

Timestamp of last successful registration from this vCenter.
ppid*xsd:string

Platform Provisioning ID (PPID). Hex-encoded representation of the the PPID (Platform Provisioning ID), returned as the response to a successful registration request. This field is populated only on the vCenter through which the host has been registered.
registrationUrl*xsd:string

SGX host registration URL. Unset if SGX registration status is not applicable or in case of an internal error.
status*xsd:string

SGX host registration status. Valid values come from RegistrationStatus enum. Set, except in case of an internal error.
type*xsd:string

SGX host registration type. Valid values come from RegistrationType enum. Unset if SGX registration status is not applicable, complete, or in case of an internal error.
